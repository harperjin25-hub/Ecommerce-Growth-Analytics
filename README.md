# E-commerce Growth & Conversion Analytics

This project analyzes end-to-end e-commerce performance using SQL, focusing on
marketing attribution, conversion funnel analysis, and customer behavior.

## Tools & Techniques
- SQL (MySQL)
- Funnel Analysis
- Marketing Attribution
- A/B Testing
- Cohort & Retention Analysis

## Key Analyses
- Channel efficiency and marketing attribution
- Session-level conversion funnel and drop-off analysis
- Landing page and checkout A/B test evaluation
- Product performance and refund rate analysis
- New vs returning user behavior

## Key Takeaways
- Identified high- and low-efficiency acquisition channels for budget prioritization
- Exposed primary conversion drop-off stages in the user journey
- Quantified potential UX-driven conversion uplift through A/B test analysis
